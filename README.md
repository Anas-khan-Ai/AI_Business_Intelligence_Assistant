# 🤖 AI Business Intelligence Assistant

An AI-powered Business Intelligence Dashboard built using Python and Streamlit. This project helps businesses analyze sales, generate insights, forecast future trends, and make data-driven decisions.

---

# 🚀 Features

- 📊 Interactive Dashboard
- 💰 Business KPI Dashboard
- 📈 Sales Forecasting
- 🤖 AutoML Prediction
- 🛒 Market Basket Analysis
- 📉 Profit Leakage Analysis
- 👔 AI Executive Dashboard
- 🗺️ Interactive Map Dashboard
- 💡 AI Business Insights
- 🚨 AI Anomaly Detection
- 🎯 What-If Analysis
- 🧹 AI Data Cleaning Assistant
- 📄 Smart PDF Report
- 📊 Excel Report Generator

---

# 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Matplotlib
- Google Gemini AI

---

# 📂 Supported Files

- CSV
- Excel (.xlsx)

---

# ▶️ Installation

```bash
pip install -r requirements.txt
```

Run the project

```bash
streamlit run app.py
```
